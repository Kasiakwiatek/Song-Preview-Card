# 🎵 Song Preview Card

A clean and minimal **song preview component** built as part of the INVO Academy learning path.  
The project focuses on structured HTML, modular SCSS, and a visually balanced UI.

## 📬 Live Preview  
[![Live Demo](https://img.shields.io/badge/Live%20Demo-GitHub%20Pages-blue?style=flat-square)](https://kasiakwiatek.github.io/Song-Preview-Card/)

## ✨ Features
- Minimal and modern UI  
- Responsive layout  
- SCSS architecture with variables and partials  
- Lightweight and fast to load  

## 🔧 Tech Stack
- **HTML5**  
- **SCSS**  
- **JavaScript** (minimal)
